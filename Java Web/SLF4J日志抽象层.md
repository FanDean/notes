# SLF4J日志

SLF4J(Simple logging Facade for Java)不是一个真正的日志实现，而是一个抽象层（ [abstraction layer](http://javarevisited.blogspot.com/2010/10/abstraction-in-java.html)），它允许你在后台使用任意一个日志类库。



SLF4J使你的代码独立于任意一个特定的日志API。
